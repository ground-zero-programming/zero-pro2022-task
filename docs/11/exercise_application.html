発展問題：画像をリサイズし，保存するインスタンス関数を実装する#

演習4で作成した AnimalImageクラスを継承し，画像をリサイズした後，保存する機能を持つ AnimalEditor クラスを作成せよ．ただし，画像のリサイズと保存は以下のコードを利用されたい．

画像の保存

os.makedirs('./output', exist_ok=True) # 画像を保存するディレクトリを作成
resized_img_path = f'./output/resized_{name}.png' # 画像を保存するファイルを指定
img.save(resized_img_path) # PILを利用して画像を保存

画像のリサイズ

画像のリサイズとは画像の大きさ（解像度ともいう）を変更する処理である．以下のコードでimgを高さと幅が \(100 \times 100\)のサイズに変更できる．

img = img.resize((100,100))
