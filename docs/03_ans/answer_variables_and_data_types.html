第3回：変数とデータ型（解答）#

担当：相澤宏旭

演習1：変数の代入#

整数11を変数aに代入せよ

# 解答
a = 11

演習2：変数の出力#

print文を利用して変数aの値を出力せよ

出力例

11

# 解答
print(a)

11

演習3：変数の型の確認#

変数aのデータ型をtype関数を利用して確認せよ

出力例

<class 'int'>

# 解答
print(type(a))

<class 'int'>

演習4：数値データの演算#

演習4-1：数値データの和#

整数値13が格納された変数aと整数値4が格納された変数bをそれぞれ定義し，aとbの足し算の結果を出力せよ

出力例

17

# 解答例 1
a = 13
b = 4
print(a + b)

17

# 解答例 2
a = 13
b = 4
c = a + b
print(c)

17

演習5：文字列#

演習5-1：文字列の代入と出力#

文字列Ground Zero Programming!が格納された変数sを定義し，変数の中身とデータ型を出力せよ

出力例

Ground Zero Programming!
<class 'str'>

# 解答
s = 'Ground Zero Programming!'
print(s)
print(type(s))

Ground Zero Programming!
<class 'str'>

演習5-2：文字数の取得#

演習5-1で定義した変数sの文字数を出力せよ

出力例

24

# 解答
s = 'Ground Zero Programming!'
print(len(s))

24

演習5-3：文字列のインデキシング#

演習5-1で定義した変数sについてインデキシングを利用してG，Z，P，!を出力せよ

出力例

G
Z
P
!

# 解答
s = 'Ground Zero Programming!'
print(s[0])
print(s[7])
print(s[12])
print(s[-1])

G
Z
P
!

演習5-4：文字列のスライシング#

演習5-1で定義した変数sについてスライシングを利用してZeroを出力せよ

出力例

Zero

# 解答
s = 'Ground Zero Programming!'
print(s[7:12])

Zero

演習6：リスト#

演習6-1：リストの作成#

以下の4つの値を持つリスト l を作成し，lの値とデータ型を出力せよ

値

27.4
29.0
33.8
30.1

出力例

[27.4, 29.0, 33.8, 30.1]
<class 'list'>

# 解答
l = [27.4, 29.0, 33.8, 30.1]
print(l)
print(type(l))

[27.4, 29.0, 33.8, 30.1]
<class 'list'>

演習6-2：リストの要素数の計算#

演習6-1で定義したリストlの要素数を出力せよ

出力例

4

# 解答
print(len(l))

4

演習6-3：リストのインデキシング#

演習6-1で定義したリストlに含まれる値33.8をインデキシングを利用して出力せよ

出力例

33.8

# 解答
print(l[2])

33.8

演習6-4：リストのスライシング#

演習6-1で定義したリストlに含まれる2つの値29.0と33.8をスライシングを利用して出力せよ

出力例

[29.0, 33.8]

# 解答
print(l[1:3])

[29.0, 33.8]
